Apple Quality Assessment
(Machine Learning)

This project is a fruit quality assessment system using Machine Learning. This system is designed using the python programming language using the SVM Algorithm for classification and GLCM for feature extraction. This project succeeded in determining the quality of apples and classifying whether the apples were fresh or rotten. This system has a classification accuracy of 84.167%. This project was made using Google Collab
